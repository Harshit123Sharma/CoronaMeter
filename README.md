# CoronaMeter App
It is an App used to get info on Covid-19 Cases. It is deployed on Vercel :- [Corona Tracker](https://corona-meter.vercel.app/)

## Installation
---
To run this project, install it locally using npm :-
```bash
$ cd CoronaMeter
$ npm install
$ npm start
```
